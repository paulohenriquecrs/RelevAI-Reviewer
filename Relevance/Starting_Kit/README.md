### Virtual env
Follow these instructions to create and run your notebook in a conda env.

1. Create conda env

```bash
conda create -n relevance python=3.9
```

2. Activate env

```bash
conda activate relevance
```

3. Install required packages

```bash
pip install \
numpy \
pandas \
matplotlib \
scikit-learn \
sentence-transformers \
transformers \
torch \
ipywidgets
```
